소감

  • (현우) 모델링 부분만 신경썼었는데 저작권을 비롯한 법률적인 문제에 대해 생각해 볼 수 있어서 좋았다.

  • (준배) 오늘은 어려운 부분이 많았던 것 같다.

    • 법적인 부분이다 보니 생각지도 못한 점이 많았던 것 같다.
    • 교양 수업을 비슷한 것을 들은 적이 있었는데 그때도 애매한 점이 많다고 느꼈다.
    • 나중에 회사에 들어갔을 때는 법률 팀에 확실하게 논의해야 하겠다는 생각이 들었다.

  • (준호) 현재 필요하기보단 나중에 찾아봐야지 하는 지식이었다.

    • NLP도 관심분야는 아니지만 재밌게 들었다.

  • (상진) 공공 데이터 포럼에서도 느꼈지만, 정보를 모으고 2차 저작물을 만들 때 개인의 동의가 필요하다는 것을 다시 한 번 느꼈다. 현업에서 꼭 필요한 부분이기 때문에 오늘은 그냥 넘어가지만 정리하고 넘어가야 한다는 생각이 들었다.

  • (지원) 관심있던 NLP지만 기술적인게 많아서 정리해서 다시봐야겠다 생각.

오혜연 마스터 : AI & Ethics

  • BIAS -> 저번 피어세션에서 했던 얘기와 유사
    • 통계학에서 Simpson’s paradox / 남학생들이 선호하는 학과가 쉬운 학과가, 여학생들이 선호하는 학과에는 어려운 학과에 지원 -> 전체적으로는 남학생의 합격률이 높지만, 학과별로 보면 그렇지 않다.
    • 데이터에 주관적인 의미부여가 있으면 문제가 될 것 같다.
    • 통계나 데이터를 더 잘 이해하는 사람일수록 조작하기 쉬운 것 같다.
    • 조작도 문제지만, 선의를 가지고 했을 때도 편향된 결과가 나올 수도 있다.
    • 진입장벽이 높다 보니까 조작을 판별하기 어려운 경우가 많지 않을까?
  • 인공지능 지식을 갖고 있는 사람과 그렇지 않은 사람의 격차?
    • 사회 내 다른 분야에서도 마찬가지
    • 예 – 넥슨 게임 사태에서 확률 조작 -> 법으로 확률 정함
  • 윤리로 인한 규제로 인해 다른 나라와 기술 격차가 발생할 수도 있다고 생각하는데, 이에 대한 대책이 있을까요?
    • 생명공학에서 생명윤리 문제와 비슷한 것 같다.
    • ‘최소한의 선이 있다’
  • 인공지능이 발전해서 사람처럼 만들어진다고 해도, delete 누르면 죽이는 건가요?
    • 영화/드라마 – 아이로봇, 휴먼스
  • 윤리로 인한 규제로 인해 다른 나라와 기술 격차가 발생할 수도 있다고 생각하는데, 이에 대한 대책이 있을까요? (국내/국제)
